# Decidí enfocarme en React.js para aportar desde el primer día.

Para fortalecer esta especialización, realicé un curso completo de React.js. Al terminar el proceso, organicé y desplegué todas mis prácticas en un monorepo estructurado profesionalmente. Este abarca desde los fundamentos de React hasta conceptos clave como hooks, routing, manejo de formularios y buenas prácticas de desarrollo moderno.

Hice este proyecto por iniciativa propia, mejorar mi capacidad de aprendizaje autónomo y cuidar mi compromiso con crecer como React Developer, además de llevar un paso más allá mi educación formal como Técnico en Desarrollo de Software en el SENA y estar realmente preparado para aportar desde el primer día en un equipo de desarrollo React profesional.
